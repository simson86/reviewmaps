# ReviewMaps

네이버/구글 지도를 불러와 방문한 장소의 **사진, 리뷰, 그리고 나만의 경험을 일기처럼 기록**할 수 있는 앱입니다.

## 주요 기능 (계획)

- 🗺️ 네이버/구글 지도 연동 및 장소 검색
- 📍 방문한 장소를 지도에 핀으로 기록
- 📷 사진 업로드 및 갤러리
- ✍️ 장소별 리뷰 및 일기 작성
- 🔎 기록한 장소 다시 찾아보기

## 기술 스택

> 추후 확정 (예: 프론트엔드 프레임워크, 지도 API, 백엔드/DB 등)

## 시작하기

```bash
# 의존성 설치 (스택 확정 후)
# npm install

# 개발 서버 실행
# npm run dev
```
